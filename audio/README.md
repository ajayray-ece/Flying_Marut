# Audio Files Directory

Place your audio files here:

- **background.mp3** - Background music that loops during gameplay
- **jump.mp3** - Sound effect when the player jumps/flies up
- **coin.mp3** - Sound effect when collecting coins
- **gameover.mp3** - Sound effect when the game ends

## Audio Format Recommendations

- **Format**: MP3 is recommended for best browser compatibility
- **Bitrate**: 128 kbps is sufficient for game audio
- **Duration**: 
  - Background music: 30-60 seconds (will loop)
  - Sound effects: 0.5-2 seconds

## Free Audio Resources

You can find free game audio at:
- Freesound.org
- OpenGameArt.org
- Zapsplat.com
- Incompetech.com (for background music)

The game will work without these files, but audio won't play.
